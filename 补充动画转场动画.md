# 转场动画

---

##对于UINavigationController添加转场动画
####步骤：
#####1、创建CATransition对象
```
CATransition * transition = [CATransition animation];
```
#####2、添加动画时间
```
transition.duration = 10;
```
#####3、动画属性
######通过函数设置
```
[transition setTimingFunction:[CAMediaTimingFunction functionWithName:kCAMediaTimingFunctionLinear]];
```
